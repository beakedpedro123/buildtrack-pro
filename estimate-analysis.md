## Pattern Analysis — Comparing Both Estimates

### Consistent Structure
Both estimates follow the same template:
1. **Header**: Company logo, estimate #, date, client info
2. **Project info**: Project name, plan reference
3. **Scope of Work**: Standard boilerplate + project-specific items
4. **Exclusions**: Standard exclusions
5. **Square footage breakdown**: By level/area
6. **Line item table**: Description, Qty, Unit Price, Total
7. **Terms & Conditions**: 8 standard clauses
8. **Signature line**

### Common Line Item Categories
| Category | Hardy | Gruett | Pattern |
|---|---|---|---|
| Structural framing | $29,205 (71.1%) | $177,120 (39.4%) | Always the largest single item |
| Steel/Timber install | $4,250 (10.4%) | $174,150 (38.7%) | Varies hugely by project complexity |
| Specialty work | - | $48,222 (10.7%) | Drop ceilings, shade pockets — project-specific |
| Nails & fasteners | $3,000 (7.3%) | $17,600 (3.9%) | Scales with project size |
| Crane & equipment | $4,600 (11.2%) | $33,000 (7.3%) | Scales with project size |

### Pricing Insights
- **Hardy**: 3,903 sqft at ~$10.52/sqft — simpler residential framing
- **Gruett**: 8,856 sqft at ~$50.82/sqft — complex cabin with steel, drop ceilings
- The per-sqft rate varies dramatically based on complexity (steel work is the major driver)
- **Nails/fasteners**: ~$0.77/sqft (Hardy) vs ~$1.99/sqft (Gruett) — scales with complexity
- **Crane**: ~$1.18/sqft (Hardy) vs ~$3.73/sqft (Gruett) — heavier equipment for steel
- **Change order rate**: $50-55/man hour — consistent range
- **Deposit**: 15-30% — varies by relationship/project size
